# CalSleek

CalSleek is a modern calorie and nutrition tracking application designed to help users achieve their fitness and health goals through simple and intuitive food logging.

Whether you're trying to lose weight, gain muscle, or maintain your current physique, CalSleek provides an easy way to monitor your daily calorie intake and macronutrients.

##  Features

* User authentication and authorization
* Personalized calorie and macro goals
* Food tracking and meal logging
* Daily nutrition overview
* Weight tracking and progress monitoring
* Historical nutrition data
* Responsive and user-friendly interface
* Modern clean design

##  Tech Stack

### Backend

* Java Spring Boot
* Spring Security
* Spring Data JPA
* PostgreSQL
* Maven

### Frontend

* Flutter
